# LogicTask1_Feb2026
Logic Building Task 1 – Includes User Login Check, Pass/Fail Analyzer, Data Cleaner, Message Length Analyzer and Error Message Detector.
All tasks are implemented in Python and focus on basic logic, data processing and analysis exercises.

**Tasks Covered:**

User Login Check – Validate username and password input.

Pass / Fail Analyzer – Analyze student marks and count pass/fail.

Simple Data Cleaner – Standardize user names by removing spaces and converting to lowercase.

Message Length Analyzer – Analyze message sizes and flag messages longer than 10 characters.

Error Message Detector – Count “ERROR” entries in system logs.

**Real-World Applications:**

Authentication systems

Academic evaluation systems

Data preprocessing for analysis

Text filtering and validation

Monitoring and log analysis systems

**Environment / Tools Used:**  
- Jupyter Notebook
  
- Python (Pyodide)
